## Getting started
1. `git clone https://github.com/alexcarpenter/material-jekyll-theme.git`
2. `cd material-jekyll-theme`
3. Configure the `_config.yml` file as needed
4. `jekyll serve`

## Options
Customize your options within the `_config.yml` file.

+ Theme
  - Green
  - Blue
  - Orange
  - Purple
  - Grey
+ Fixed Navigation
  - True
  - False
